### 반려견 이물섭식 탐지 어플


- 이물섭식을 탐지하는 딥러닝 모델을 학습시킨다.
- 앱을 통해 반려견의 시야를 실시간으로 확인한다.
- 반려견의 시야 영상을 실시간으로 모델에 전송하여 이물섭식을 탐지한다.
- 이물섭식이 탐지될 경우 반려견주에게 알람을 전송한다.
